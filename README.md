# Bi-Dom Backend

The backend application managing the Bi-Dom system with real-time broadcasting.

## Key Features
- Real-time events with Laravel Echo and Pusher
- API endpoints for the client app
- Modern asset bundling with Vite

## Tech Stack
- Laravel
- Laravel Echo
- Pusher JS
- Vite
- Tailwind CSS
